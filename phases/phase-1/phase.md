# Phase 1 — Foundation

## Goal
Project setup, input system, player controller (movement, dodge roll), camera system, basic room with placeholder art. Player can move around a test room and dodge.

## Key Deliverables
- Unity project configured with 2D URP, Input System, Tilemap packages
- Singleton base class and GameManager
- GameEvents static class for event-driven communication
- Player controller with 8-directional movement using new Input System
- Dodge roll with i-frames and cooldown
- Camera follow system
- Test room with placeholder tilemap art
- Basic project folder structure (Scripts/, Prefabs/, ScriptableObjects/, Scenes/)

## Entry Criteria
- Atlas setup approved
- Code repository initialized

## Exit Criteria
- Player can move in 8 directions with analog stick or WASD
- Player can dodge roll with visible i-frames
- Camera smoothly follows player
- Test room has walls, floor, and collision boundaries
- Project compiles without errors
- Folder structure matches architecture conventions

## Dependencies
- None (first phase)
